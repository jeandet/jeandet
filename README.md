## Alexis Jeandet

R&D Engineer at [LPP](https://lpp.ip-paris.fr) / CNRS — building software for space physics, from FPGA flight code to Python data APIs.

### What I build

**[SciQLop ecosystem](https://github.com/SciQLop)** — an open-source stack for accessing, analyzing, and visualizing in-situ space plasma data:

| Layer | Project | What it does |
|-------|---------|-------------|
| Data access | [Speasy](https://github.com/SciQLop/speasy) | Unified Python API — 65k+ products from AMDA, CDAWeb, CSA, SSCWeb |
| CDF I/O | [CDFpp](https://github.com/SciQLop/CDFpp) | C++ CDF library, up to 4 GB/s reads, Python bindings |
| Plotting | [SciQLopPlots](https://github.com/SciQLop/SciQLopPlots) | C++20/Qt6 engine for large non-uniform datasets |
| Desktop app | [SciQLop](https://github.com/SciQLop/SciQLop) | Interactive browser for space mission data |
| Collaboration | [cocat](https://github.com/SciQLop/cocat) | CRDT-based real-time event catalog co-editing |
| Linting | [AstraLint](https://github.com/SciQLop/AstraLint) | Validates space data files against ISTP/PDS4 standards |

Also: [NeoQCP](https://github.com/SciQLop/NeoQCP) (next-gen QRhi-based plotting engine)
